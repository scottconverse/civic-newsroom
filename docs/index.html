Civic Newsroom

An open-source toolkit for AI-powered civic transparency in any community

Since 2005, more than 2,500 U.S. newspapers have closed. City councils still meet. Budgets still get passed. Contracts still get awarded. But in most communities, nobody is reading any of it.

Civic Newsroom is a complete, free editorial pipeline that turns your city's public records into accessible civic reports — using AI tools you already have access to. Nine coordinated agents handle everything from signal detection to publication-ready copy, with built-in safeguards against hallucination, plagiarism, and one-sided analysis.

The Pipeline

Nine agents, each handling a different stage of the civic reporting process.

Agent 1
News Aggregator
Scans your city's public records, agendas, and government portals. Produces 15–25 prioritized story leads daily with source conflict resolution and skip-logic for missing packets.
Agent 2
Story Expansion
Converts the strongest leads into 400–800 word publication-ready stories with pre-expansion verification gate, standardized confidence scoring, and revision pathways for suppressed stories.
Agent 3
Black Desk
Speculative signal radar. Looks for the dog that didn't bark — absences, anomalies, and fiscal stress patterns. Includes defined cluster thresholds and tiered source references.
Agent 4
Dark Signal Desk
Mandatory adversarial four-gate verification with reporting handoff. Verified signals route to the daily pipeline or standalone research. Requires searching 3+ platforms within a 90-day window with documented query variations.
Agent 5
Integrity Checker
Universal final gate — every story from any prompt must pass this five-part audit before publication. Tests links, audits claims against Tier A sources, flags missing voices, and provides hold/publish/correct guidance with CRITICAL/MINOR severity.
Agent 6
First Amendment Counsel
Legal guidance on press freedom issues — defamation, SLAPP suits, source protection, public records access. Includes plain-language definitions of legal concepts like “actual malice.”
Agent 7
Plain-Language Translator
Rewrites complex civic reporting into clear, accessible language without changing the facts. Now includes failure-path explanations for conditional policies.
Agent 8
Civic Grounding Protocol
Anti-plagiarism safeguards with primary source clarifications. Ensures reporting traces back to primary public records, not other journalists’ work, with revision pathways for suppressed stories.
Agent 9
Story Research & Writing
Standalone research-and-write agent. Give it a topic, it runs a five-stage sequence with attribution templates and source conflict handling. Must pass the Integrity Checker before publication.

Built-In Safeguards

AI-assisted civic reporting fails in specific, predictable ways. This system prevents each of them architecturally.

Adversarial Verification

A mandatory four-gate protocol forces the system to search at least 3 platforms within a 90-day window, using 3+ query variations, and document every search attempt before any contested claim can advance. AI will produce one-sided analysis without this gate. The protocol makes that failure mode structurally impossible.

Anti-Plagiarism Architecture

The Civic Grounding Protocol ensures every claim traces back to primary public records. Other outlets' coverage can tell you what to look for. It can never be your source.

Suppression Ledger

Every killed story gets a record: why it was killed, what was missing, and what would make it publishable. Nothing disappears silently. Suppressed stories include a defined revision pathway for reopening when evidence appears.

Integrity Auditing

The Integrity Checker is the universal final gate for the entire system. Every story — whether from the daily pipeline or standalone research — must pass its five-part independent audit before publication. No prompt can self-certify its own output as publication-ready.

Source Tier System

A shared reference defines four confidence tiers — from primary government records (Tier A) through community signals (Tier C) to experimental sources (Tier D). Every agent in the pipeline uses the same definitions, ensuring consistent sourcing standards across the entire editorial process.

Get Started in 30 Minutes

  1. Clone or download the repository from GitHub
  2. Find your city's key URLs: council agenda portal, official news page, school district site
  3. Fill in the News Aggregator prompt with your city's info
  4. Paste it into Claude or Gemini (both free; requires a large context window — ChatGPT free tier is too small)
  5. Read the output — you'll get 15–25 story leads from your city's public records
  6. Expand the best leads into full stories using the Story Expansion agent
  7. Run every story through the Integrity Checker before publishing
